# Eyelink-Pupillometry-Spanish-Subjunctive-Using-EB

This repository contains the materials and implementation of a pupillometry-based psycholinguistic experiment investigating how bilingual speakers process Spanish subjunctive morphology in real time.

The experiment is built using Experiment Builder and the EyeLink 1000 Plus system, integrating auditory stimuli, trial-level experimental control, and eye-tracking data collection. The project includes stimulus datasets, experimental configuration, and preprocessing steps for combining behavioral and physiological data.

This is an ongoing research project focused on testing predictions from the Bottleneck Hypothesis and examining differences between native speakers, heritage speakers, and L2 learners.
